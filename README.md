# Meta_Ads_Performance_Data_Analysis

## Meta Ads ROAS Validation with Python

This project validates Meta (Facebook & Instagram) Ads performance by reconciling platform-reported metrics with WooCommerce backend sales data.

Instead of relying solely on Ads Manager dashboards, the analysis calculates true ROAS using actual order revenue to identify attribution differences.

# What This Project Does

Loads Meta Ads and WooCommerce data

Cleans and validates data (dates, revenue, order status)

Calculates:

Website backend revenue

Meta-reported revenue

Total ad spend

Website ROAS vs Meta ROAS

Exports a clean ROAS summary report for business use
